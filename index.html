Apollo

About

Apollo Music Application is a one-stop-shop for tour schedule, twitter feed, and latest news articles on any music artist. Simply search for any music artist by name and Apollo will pull the results into a user friendly interface.

Features

  • Search Feature- quick and easy searching for any music artist. One search will pull in all of the below information, eliminating the need for multiple searches.
  • Twitter Feeds- allows the user to see the up-to-date Twitter feed of their favorite artists.
  • Touring Information- allows users to browse the latest touring information, as well as links to purchase tickets.

My Role

Apollo was a passion project of mine. I realized the need for a centralized source of informaiton relating to music artists. Taking ownership of Apollo, I was responsible for developing the entire application start to finish. Apollo is a Ruby on Rails application, utilizing Backbone.js and Underscore.js for the Front-End.

Servus

About

Servus is a fully functional POS application that allows for servers to organize and track their orders, as well as manage their tables.

Features

  • Table Tracking - Allows for servers to create and track table information, such as table number and number of guests. The Servus Interface also allows for quick and easy access to order history for each table.
  • Order Selection - Allows for servers add orders to specific tables by first selecting the table, and then selecting the food item. Selected food item will be highlighted in green for quick recognition.
  • Customizable Menu - The customizable menu allows managers to quickly update the menu that servers will have access to. Food items that are no longer on the menu may be deleted.
  • Adding Menu Items - Managers can add new food items to the menu, through entering the necessary information, including price, name, type of cuisine, and any allergens the food may contain.

My Role

>Servus was a personal project of mine to test out a Sinatra (Ruby) application. I was responsible for both Fron-End and Back-End development. Backbone.js and Underscore.js were used in the client-side development.

Blocktrotter

About

Blocktrotter is a application that helps visitors and residents better explore the dozens of neighborhoods in the city of Chicago. With Yelp and Foursquare recommendations, Blocktrotter gives users the best options for their trip.

Features

  • Search Feature - allows users to search for specific places within each neighborhood.
  • Foursquare / Yelp Results - search function returns both the Yelp and Foursquare top results for that search query. This allows gives users multiple options as well as the ability to compare places across both Yelp and Foursquares ratings.

My Role

My role in the development of Blocktrotter was Front-End developer. I was tasked with managing the client-side rendering of our data, client-side validaiton, as well as styling.
